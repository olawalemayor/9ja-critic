# About Project

The project is divided into two parts, the front end and the backend

## To migrate to the frontend

Run command:

```bash
cd client
```

### To start dev mode

```bash
npm run dev
```

### To build the app

```bash
npm run build
```

### To preview the app

```bash
npm run preview
```

## To migrate to the backend

Run command:

```bash
cd server
```

## To install the node packages

Run command:

```bash
npm install
```

## Backend

### To start server

Run command:

```bash
npm start
```
